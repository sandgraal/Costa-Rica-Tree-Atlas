# Usage Policy

## Costa Rica Tree Atlas - Ethical Usage Guidelines

This document outlines the ethical usage guidelines for the Costa Rica Tree Atlas project, supplementing the dual-license structure (AGPL-3.0 for code, CC BY-NC-SA 4.0 for content).

## Mission Statement

The Costa Rica Tree Atlas exists to:

- Educate people about Costa Rican biodiversity
- Support conservation efforts
- Provide accessible scientific information
- Promote environmental stewardship

## Acceptable Uses

### ✅ Encouraged Uses

The following uses are explicitly encouraged and supported:

1. **Educational Purposes**
   - Classroom instruction and teaching materials
   - Student research projects
   - Educational presentations and workshops
   - Science communication

2. **Research & Conservation**
   - Scientific research and data analysis
   - Conservation planning and assessment
   - Biodiversity monitoring programs
   - Environmental impact studies

3. **Non-Profit Organizations**
   - Conservation NGO materials
   - Community education programs
   - Public awareness campaigns
   - Environmental advocacy (aligned with conservation goals)

4. **Personal Use**
   - Field identification guides
   - Nature photography reference
   - Personal learning and exploration
   - Hobby naturalist activities

5. **Open Source Development**
   - Contributing to the project
   - Creating derivative educational tools
   - Improving accessibility
   - Translating content to additional languages

## Prohibited Uses

### ❌ Explicitly Prohibited

The following uses are NOT permitted under the project licenses:

1. **Commercial Exploitation**
   - Selling tree data or content without permission
   - Incorporating content into commercial apps without agreement
   - Using images in commercial products or advertising
   - Paywalled access to derivative works

2. **Harmful Industries**
   - Use by extractive industries (logging, mining) for exploitation purposes
   - Agricultural expansion that threatens forest ecosystems
   - Land development projects that harm biodiversity
   - Any use that facilitates environmental degradation

3. **Misrepresentation**
   - Claiming ownership of content
   - Removing attribution or copyright notices
   - Misrepresenting the source of information
   - Using content in ways that contradict conservation goals

4. **Competitive Services**
   - Creating commercial competitors without contributing back
   - Proprietary forks that don't share improvements
   - Closed-source applications using project data for profit

## Commercial Licensing

### Requesting Permission

If your use case doesn't fit the above categories, or you need a commercial license, please:

1. **Contact the maintainer** via GitHub Issues or Discussions
2. **Describe your use case** in detail
3. **Explain how your use aligns** with the project's conservation mission
4. **Propose a contribution** back to the project or conservation efforts

### Evaluation Criteria

Commercial license requests will be evaluated based on:

- Alignment with conservation and education goals
- Benefit to the broader community
- Contribution back to the project
- Environmental and social impact
- Attribution and recognition practices

### Examples of Potential Commercial Licenses

We may grant commercial licenses for:

- Eco-tourism operators creating ethical, educational materials
- Conservation-focused mobile apps that share revenue or improvements
- Publishing houses creating educational books about Costa Rican nature
- Documentary filmmakers working on conservation themes

## Attribution Requirements

### Required Attribution

When sharing or adapting content, you must:

1. **Credit the project**: "Costa Rica Tree Atlas (https://github.com/sandgraal/Costa-Rica-Tree-Atlas)"
2. **Include copyright notice**: "Copyright (c) 2024-present sandgraal"
3. **Link to licenses**: Provide links to LICENSE and LICENSE-CONTENT.md
4. **Indicate modifications**: Clearly mark any changes you've made

### Attribution Format

```
Source: Costa Rica Tree Atlas
Copyright (c) 2024-present sandgraal
License: AGPL-3.0 (code) and CC BY-NC-SA 4.0 (content)
URL: https://github.com/sandgraal/Costa-Rica-Tree-Atlas
[Modified/Unmodified]
```

## Data Integrity

### Responsibility for Accuracy

Users of this data must:

- Verify information for critical applications
- Cite scientific sources when available
- Report errors or inaccuracies through GitHub Issues
- Not present data as medical, legal, or safety advice without proper disclaimer

### Toxicity & Safety Information

Safety information is provided for educational purposes only:

- Always consult professionals for safety concerns
- Information may not be comprehensive
- Local poison control should be the primary resource for emergencies
- Project maintainers are not liable for harm resulting from use of information

## Contributing Back

### Open Source Ethos

We strongly encourage users to:

- Share improvements back to the project
- Contribute new tree profiles
- Report bugs and suggest enhancements
- Help with translations
- Contribute photographs (with proper licensing)

See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines on contributing.

## Enforcement

### License Violations

Violations of these terms may result in:

1. Request to cease and desist
2. Removal of access to project resources
3. Public notice of violation
4. Legal action if necessary

### Reporting Violations

To report misuse of project content:

1. Open a GitHub Issue with subject "License Violation Report"
2. Include evidence and description of violation
3. Maintainers will investigate and respond

## Questions & Clarifications

If you're unsure whether your use case is permitted:

1. **Check the licenses**: Read [LICENSE](LICENSE) and [LICENSE-CONTENT.md](LICENSE-CONTENT.md)
2. **Review examples** in this document
3. **Ask in GitHub Discussions**: We're happy to provide clarification
4. **When in doubt, ask**: Better to confirm than violate terms

- Adapt to changing circumstances
