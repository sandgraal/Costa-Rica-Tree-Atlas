# Comparison Images

This directory contains images specific to tree comparison guides.

## Directory Structure

```
comparisons/
  {comparison-slug}/
    feature-1.jpg
    feature-2.jpg
    ...
```

## Naming Convention

- Use the comparison slug as the subdirectory name
- Use descriptive names for feature images: `leaf-comparison.jpg`, `bark-detail.jpg`
- Include both species in filename when showing both: `mango-espavel-side-by-side.jpg`

## Image Requirements

- **Format**: JPG or WebP preferred
- **Minimum dimensions**: 800x600px
- **Aspect ratio**: 4:3 preferred for consistency
- **File size**: Optimize to under 500KB per image

## Attribution

All images must have attribution tracked. Update `attributions.json` in this directory when adding new images.
